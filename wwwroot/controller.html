MeowWorld - ASP.NET Webapp/Webservices on OSCP Demo

Uth (Red Hat UKISSA)

Usage:
Enter the webservice url (root as above)/cat/all to get all the cats in the static version.
Enter the webservice url (root as above)/cat/{name} to get the static noise for the cat via webservice.
Click the 'Play with the Horde' to create and add cats to your own horde.
